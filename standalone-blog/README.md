# EGreet Blog - Standalone Deployment

This is a standalone blog that can be deployed separately from the React app.

## Files:
- `index.html` - Main blog page
- `css/blog-styles.css` - Blog styling
- `js/blog.js` - Blog JavaScript with 16 blog posts
- `post/` - All blog post files (26 total)
- `sitemap.xml` - Blog sitemap for SEO
- `robots.txt` - Search engine instructions

## Deployment:
1. Upload this entire folder to your server at `/blog/`
2. The blog will be accessible at `https://egreet.in/blog/`
3. Submit `https://egreet.in/blog/sitemap.xml` to Google Search Console

## Features:
- 16 new blog posts + 10 existing posts
- Search and filter functionality
- SEO optimized
- Responsive design
- No React dependency
